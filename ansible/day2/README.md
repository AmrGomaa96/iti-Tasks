# Project OverView
### using terraform to build the architect & ansible to configure the private instaces.

#

## Terraform
### Run terraform files using the following command 
```
terraform plan 
terraform init
terraform apply
```
#
## Ansible 
### Run ansible :
```
ansible-playbook playbook.yml -i inventory.txt
```
